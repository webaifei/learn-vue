### vue是什么？
> 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用
1. 渐进式，构建用户界面：分层架构，解耦
2. 组件化： 保持简单、复用
3. 高效：虚拟DOM
